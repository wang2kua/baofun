# BaoFun 设计文档

日期:2026-06-29
状态:已确认,待实现

## 一句话定位

把一台旧安卓手机变成一个"会发声、对触摸有反应的极暗护眼玩具",给约 10.5 个月的宝宝玩。屏幕几乎全黑以保护眼睛,靠**声音 + 触觉(震动)+ 极小的暖色微光**提供因果反馈。

## 重要前提(必须摆正的预期)

主流儿科建议(AAP / WHO)是 18-24 个月以下婴儿尽量不接触屏幕。本 App **不是**"可以放心天天玩的护眼神器",而是"当大人需要腾出手时,一个远比随便丢个视频给宝宝要好的替代品"。真实陪玩(积木、绘本、躲猫猫)永远优于任何屏幕。所有设计决策都服务于"既然要用,就尽量降低伤害"。

## 技术栈

- **Kotlin + 原生 Android**,无第三方框架。
- 单 Activity,全屏锁定。
- 最低支持 Android 6.0 左右(适配旧手机)。
- 音效:Android 自带 `SoundPool` / 合成乐音(不依赖外部音频文件)。
- 儿歌:`MediaPlayer` 播放打包在 `assets/songs/` 的 mp3。
- 震动:`Vibrator`。
- 设置存储:`SharedPreferences`。
- 录音存储:App 私有目录。
- **不联网、无广告、不收集任何数据**;仅申请麦克风(录音)和震动权限。

## 三个状态

1. **玩耍模式**(默认):极暗黑屏,宝宝拍/摸任意位置触发反馈。
2. **听歌模式**:全黑,循环播放内置儿歌。
3. **大人控制菜单**:平时隐藏,靠隐藏手势呼出。

## 模块设计

### 1. 玩耍模式

宝宝拍/摸屏幕任意位置时,**同时**发生三件事:

1. **微光**:触摸点出现柔和**羽化(模糊边缘)**的**暖色、低饱和**光晕(米黄/琥珀),亮度压到刚好可见,约 2 秒淡出。手指离开后回到全黑。**避免"黑底亮点"式的硬反差。**
2. **声音**:屏幕(不可见地)划分为多个区域,**不同区域对应不同音高**(类似隐形木琴/钢琴),音色柔和不刺耳。
3. **轻震动**:极短的一下震动,增强触觉因果。

**爸妈录音彩蛋**:在拍击之间**随机**触发(约每 8-10 次拍击一次,而非每次),播放预先录制的爸妈的话。

### 2. 听歌模式

- 屏幕**完全全黑**(连微光也没有,因为是听不是看)。
- 循环播放 `assets/songs/` 中的 mp3。
- 音频文件由用户自行放入(如贝乐虎儿歌,属个人合理使用);仓库内先放占位音频把功能跑通。

### 3. 护眼机制(核心)

| 维度 | 设置 |
|---|---|
| 背景 | 纯黑 `#000000` |
| 微光 | 暖色 + 低饱和 + 柔和羽化边缘 + 极低亮度 + 2 秒淡出 |
| 屏幕亮度 | 进入 App 强制拉到系统最低,退出恢复 |
| **使用时长** | **硬性定时:默认 5 分钟到点自动黑屏并停止所有反馈,需大人手势重启** |
| 使用环境 | 在大人菜单中提示:建议有微弱环境光时使用,不要在全黑房间玩 |
| 主次 | 声音 + 震动为主,微光为辅 |

### 4. 大人控制与防误触

**呼出手势**:**右上角长按 3 秒**(婴儿几乎不会精准触发,大人单手易操作)。

**控制菜单内容**(呼出后屏幕临时变亮、可读):
- 切换模式:玩耍 / 听歌
- 音量调节
- 录音管理:录爸妈的话、试听、删除
- 重新开始计时(5 分钟到点后从这里重启)
- 退出 App
- 来电提示说明(见下)

**防误触底座**:
- 锁定屏幕方向为**竖屏**。
- 屏蔽返回键 / 拦截退出,宝宝乱拍不会退出 App。
- 沉浸式全屏,隐藏状态栏和导航栏,宝宝划不出系统界面、碰不到通知。
- **来电限制**:普通 App 无法拦截真实来电(系统级限制)。菜单内放置说明,建议大人在给宝宝玩之前手动开启**飞行模式或勿扰**。

## 数据与隐私

- 录音 → App 私有目录。
- 设置(音量、定时时长等)→ `SharedPreferences`。
- 儿歌 → `assets/songs/`(用户自行放入)。
- 全程不联网、零数据收集、无广告。

## 测试策略

- **单元测试**:音区划分、定时器逻辑、录音随机触发概率等纯逻辑。
- **真机验证**:UI 与触摸反馈在用户的旧安卓手机上实际安装 apk 验证。
- 先在模拟器跑通,再产出可安装 apk。
- 搭好完整的开发-构建-测试循环:可一键编译 apk、可跑测试,迭代直至满足需求。

## MVP 范围(第一版)

包含:玩耍模式(拍击出声 + 分区音高 + 轻震动)、爸妈录音彩蛋、听歌模式、护眼(纯黑 + 强制最低亮度 + 柔和暖光 + 5 分钟自动停)、大人控制菜单、防误触底座。

## 后续迭代(明确不在第一版)

- 多点触摸和弦
- 滑动音阶(连续音)
- 躲猫猫模式(物体恒存)
- 白噪音/哄睡模式(海浪、雨声、心跳 + 定时关闭)
